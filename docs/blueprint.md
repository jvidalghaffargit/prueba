# **App Name**: ExcelSaver

## Core Features:

- Data Entry Form: User-friendly form to input invoice data (date, amount, customer, etc.).
- Data Validation: Basic input validation to ensure data integrity.
- Excel Generation: Generate and download an Excel file with the entered invoice data. The format of the excel includes the datatypes of each columns of the file.
- Column customization: Give the users the possibility to reorder or hide any column inside the file, to adapt it to different situations

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and reliability.
- Background color: Light gray (#F5F5F5) for a clean and professional look.
- Accent color: Soft green (#81C784) for positive actions and feedback.
- Body and headline font: 'PT Sans', a friendly and modern sans-serif font.
- Use clear and professional icons from a standard library like FontAwesome.
- Clean and organized layout with clear sections for data entry and file download.
- Subtle transitions and feedback animations on form submission and file generation.
